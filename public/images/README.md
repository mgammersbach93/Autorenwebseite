# Hero-Bild für die Startseite

Damit das Hero-Hintergrundbild angezeigt wird, die Datei hier ablegen:

**Dateiname:** `entdecke_deine_neuen_seiten.webp`

**Kopierbefehl (Linux/WSL):**
```bash
cp /mnt/user-data/uploads/entdecke_deine_neuen_seiten.webp public/images/
```

**PowerShell (Windows, wenn Quelle anders liegt):**
```powershell
Copy-Item "C:\Pfad\zu\entdecke_deine_neuen_seiten.webp" -Destination "public\images\"
```
